# Duck Social Network (Java)

A small social network application developed in Java using JavaFX.

The application allows users to create accounts, manage friendships, and interact inside a simple social network system.

## Project Structure

src/main/java
model – contains the main entities such as User, Friendship and Message

repository – handles database operations and data persistence

service – contains the business logic of the application

ui – JavaFX controllers and interface logic

validator – classes used to validate application data

src/main/resources
FXML files used for the graphical interface (login, user view, etc.)

## Technologies

Java
JavaFX
SQL / Database
Gradle
FXML
